Welcome to the Blog

Hey there! Welcome to the first post on this new blog system. I built this to have a place to share random thoughts, project updates, and whatever else I feel like writing about.

How This Works

The blog system is pretty simple but clever. Instead of using a database or complex CMS, it just automatically discovers HTML files in the entries folder. Each entry is a standalone HTML file named entry1.html, entry2.html, and so on.

The blog index page uses JavaScript to scan for these files and automatically display them in a nice grid layout. When you click "Read More," you get the full entry with some nice enhancements like:

  • Automatic table of contents (if there are 3+ headers)
  • Reading progress bar at the top
  • Copy buttons on code blocks
  • External link indicators
  • Smooth scrolling

Metadata

Each entry can include metadata using data attributes on the main container:

<div class="entry-container"
     data-entry
     data-title="Your Title Here"
     data-date="December 8, 2025"
     data-excerpt="A short preview..."
     data-tags="tag1, tag2, tag3">
    <!-- content here -->
</div>

What to Expect

I'll be using this space for various things:

  • Updates about projects on this site
  • Tech stuff I'm working on
  • Random thoughts about coasters, music, gear, etc.
  • Tutorials or guides if I feel like it

No schedule, no promises, just whatever feels right. That's the beauty of a personal site!

Try It Yourself

If you want to build something similar, feel free to check out the source code. It's all just HTML, CSS, and vanilla JavaScript. No build tools, no frameworks, no nonsense.

The whole system is about 150 lines of JavaScript and works entirely client-side. Pretty cool, right?

Final Thoughts

Anyway, that's it for the first post. Just wanted to get something up here to test the system and explain how it works. More interesting stuff coming soon (probably).

Thanks for reading!
